# Python send push notification test


## Start frontend app

1. enter fronted
    ```shell
      cd frontend
    ```
2. Install dependencies
    ```shell
   npm install
    ```
3. Run react app
    ```shell
   npm start
    ```

4. To see if app is running open url http://localhost:3000/


## Send notification using python

1. Install python dependencies
    ```shell
   poetry install
    ```
2. run main.py file
    ```shell
   poetry run python main.py
    ```
